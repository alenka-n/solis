# Solis

## О вертикальном прототипе

Вертикальный прототип Solis — это рабочее веб - приложение, реализующее полный стек технологий от интерфейса до бэкенда, но с фокусом на ключевом сценарии использования. В отличие от горизонтального прототипа, который покрывает много функций поверхностно, наш вертикальный прототип глубоко реализует основной пользовательский путь:

**"Эмоция → Музыка → Сохранение"**

## Технологический стек

- **Frontend**: React Native
- **Backend**: Node.js
- **Хостинг**: Yandex Cloud с Docker-контейнеризацией
- **База данных**: PostgreSQL
- **Интеграции**: API Яндекс.Музыки, AI-сервисы генерации музыки

## О проекте

Solis (от лат. "sol" — солнце) — это интерактивный дневник настроения, который преобразует ваши эмоции в персонализированные саундтреки. Больше не нужно тратить время на поиск идеальной музыки — просто отметьте, что вы чувствуете, и получите уникальное звуковое сопровождение.

## Возможности вертикального прототипа

### Отслеживание настроений
- Фиксация эмоциональных состояний
- Выбор из 8+ базовых эмоций
- Настройка интенсивности каждой эмоции с помощью слайдера
- Возможность выбирать несколько эмоций одновременно
- Автоматическое сохранение с привязкой к дате и времени

### Музыкальные рекомендации
- Автоматический поиск тематических плейлистов по вашему настроению
- Беспрепятственное воспроизведение через интеграцию с Яндекс.Музыкой
- Просмотр обложек и названий подобранных коллекций

### Звуковой дневник
- Хронологическая лента всех ваших настроений
- Привязка музыки к каждому эмоциональному состоянию
- Добавление текстовых заметок к записям для сохранения контекста

### Профиль пользователя
- Быстрая регистрация
- Сохранение истории всех записей и треков
- Управление профилем (никнейм, аватар)

## Основной пользовательский поток

1. **Старт приложения** - выбор между режимом входа и регистрации
2. **Выбор эмоций** - интуитивный интерфейс с визуальным представлением настроений
3. **Генерация контента** - поиск плейлистов и треков
4. **Прослушивание** - через плейлист или из понравившихся
5. **Сохранение** - автоматическое добавление в дневник с возможностью добавления заметок, а также отображение истории прослушиваний


## Быстрый старт

Установка
bash
Клонирование репозитория

git clone https://github.com/username/solis.git
cd solis

Установка зависимостей
npm install

Запуск
bash
npm run server

Запуск React приложения (порт 3000)
npm start
