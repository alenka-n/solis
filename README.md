# solis
О вертикальном прототипе
Вертикальный прототип Solis — это рабочее мобильное приложение, реализующее полный стек технологий от интерфейса до бэкенда, но с фокусом на ключевом сценарии использования. В отличие от горизонтального прототипа, который покрывает много функций поверхностно, наш вертикальный прототип глубоко реализует основной пользовательский путь:

"Эмоция → Музыка → Сохранение"
Технологический стек
Frontend: React Native/Flutter
Backend: Node.js/Python/Go
Хостинг: Yandex Cloud с Docker-контейнеризацией
База данных: PostgreSQL
Интеграции: API Яндекс.Музыки, AI-сервисы генерации музыки ( пу пу пу ваще)

Solis (от лат. "sol" — солнце) — это интерактивный дневник настроения, который преобразует ваши эмоции в персонализированные саундтреки. Больше не нужно тратить время на поиск идеальной музыки — просто отметьте, что вы чувствуете, и получите уникальное звуковое сопровождение.

Возможности вертикального прототипа:
Фиксация эмоциональных состояний;
Выбор из 8+ базовых эмоций;
Настройка интенсивности каждой эмоции с помощью слайдера;
Возможность выбирать несколько эмоций одновременно;
Автоматическое сохранение с привязкой к дате и времени.

Автоматический поиск тематических плейлистов по вашему настроению;
Беспрепятственное воспроизведение через интеграцию с Яндекс.Музыкой (надеюсь смогу сделать мяу мяу но не факт. если что поменяю);
Просмотр обложек и названий подобранных коллекций.


Звуковой дневник:
Хронологическая лента всех ваших настроений;
Привязка музыки к каждому эмоциональному состоянию;
Добавление текстовых заметок к записям для сохранения контекста.

Профиль пользователя:
Быстрая регистрация;
Сохранение истории всех записей и треков;
Управление профилем (никнейм, аватар).

Основной пользовательский поток:
Старт приложения - выбор между режимом входа и регистрацией;
Выбор эмоций - интуитивный интерфейс с визуальным представлением настроений;
Генерация контента - поиск плейлистов и треков;
Прослушивание - через плейлист или из понравившихся;
Сохранение - автоматическое добавление в дневник с возможностью добавления заметок, а так же отображение истории прослушиваний.
